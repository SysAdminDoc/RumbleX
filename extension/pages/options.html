Backup snapshot history

Loading…

RumbleX automatically snapshots your settings before destructive operations (import, reset, restore). Snapshots stay local. Restoring a snapshot snapshots the current state first, so the operation is itself undoable. Cap controlled by backupHistoryLimit.

    Channel notifier

    Loading…

    Polls each watched channel on a background alarm and fires an OS notification (and an optional Discord webhook POST) when the channel posts a new video or goes live. Enable channelNotifierEnabled in the Settings editor + add at least one channel below.

      Account data export

      Local-only export. Requires you to be signed in to Rumble.

      Pulls account-only surfaces (watch history, follow list) into structured JSON. The export never leaves your browser — it lands in your Downloads folder. Rumble doesn't natively offer these exports today.

      Channel archive queue

      Loading…

      Paste a Rumble channel URL (e.g. https://rumble.com/c/example) and we'll queue every video from the page for direct MP4 download. Each job picks the highest-quality direct URL via Rumble's embedJS endpoint — same path the in-page Download button uses. Queue survives extension restarts; processed at downloadConcurrency jobs/minute.

        Encrypted gist sync

        Bring-your-own GitHub gist + passphrase. We never see either.

        Cross-device settings sync via a private GitHub gist you control. Your settings are encrypted client-side with AES-GCM-256 (PBKDF2-SHA256, 200k iterations) under a passphrase only you know. Neither the passphrase nor the cleartext settings ever leave your device. RumbleX has zero server.

        Setup: create a GitHub personal access token with the gist scope at github.com/settings/tokens, paste it below, then push once. The first push creates a new private gist and saves its ID automatically.

        Rant stats

        Loading…

        Cached rants from every Rumble watch page you visited while Rant Persist was enabled. Up to 30 videos × 200 rants kept. Local-only — never uploaded.
